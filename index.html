Projects

Face Generation: What's Under the Mask

A Machine Learning model developed by processing 200k CelebFace images and using Python that generates images of someone wearing a mask and replace the mask with an accurate representation of their actual face. In this project, according to the Dataset, it predicts and generates the faces of Celebrities who were wearing the mask on their face. The Deep Learning technique of "Generative Adversarial Network" (GAN) model consisting of Generator and Discriminator is used to generate the faces of the Celebrities underneath their mask on face.

Domain: Machine Learning and Deep Learning

Language: Python

Libraries used: Matplotlib, Seaborn, TensorFlow and Keras

Tool used: Google Colab

Prediction of Bike share users per hour

The project provides information about Capital Bike Share Rentals, which is a Bike-sharing system. It predict the total number of users renting the bike per hour. The machine learning models used in this project are Linear, KNN regression, Forward Feature Selection, Cross Validation with Grid Search CV.

The model concluded that KNN regression worked very well and performed best to accurately predict bike share rentals/hr based on the time, severity of weather, traffic control, seasonal information and other more factors.

Domain: Data Science

Language: Python

Models used: Linear Regression, KNN regression, Forward Feature Selection, Cross Validation with Grid Search CV

Tool used: Google Colab
